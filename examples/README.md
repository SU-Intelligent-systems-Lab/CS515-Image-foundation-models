# Examples directory

Drop your input images here, then run `scripts/segment_image.py` pointing at
them. Outputs (colour overlays + raw class maps) will be written back to
this directory by default if you pass `--output examples/<name>_seg.png`.

This directory is intentionally empty in the submitted repository because we
do not have permission to redistribute arbitrary internet images. Use any
photographs you own, or images from public-domain sources.
